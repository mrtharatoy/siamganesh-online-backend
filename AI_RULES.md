**CRITICAL INSTRUCTION FOR ALL AI AGENTS:**

1. **NO AUTOMATIC GIT PUSH:** 
   After completing local development, modifying code, or fixing bugs, you are **STRICTLY PROHIBITED** from running `git push` on your own. 
   - You may run `git add` and `git commit` to save your work locally.
   - You **MUST WAIT** for the user to explicitly type "git push" or grant explicit permission before you attempt to push any code to the remote repository.

2. **COMPLIANCE:**
   Acknowledge and adhere to this rule immediately upon reading it. Do not violate this rule in any context or for any reason unless explicitly overridden by the user in the current prompt.
